- Clone Project
    ```
    git clone https://github.com/Luthfi98/article.git
    ```
- Pindahkan Ke Folder

		```
		c:/xampp/htdocs/
		```

		```
		c:/laragon/www/
		```

		[Article](http://localhost/article) <br>
		```
		http://localhost/article
		```
